## Cockroach DB で遊んでみる

### 始め方
```
$ docker compose up -d
```
で、ノードが３つ立ち上がる。

### 遊び方
`init_data.sh` を実行するとデータが入ります。
データ挿入クエリは `roach1` ノードに向けられています。

`bench_to_roach1.sh` をすると、 5分間ベンチマークが走ります。
`localhost:8080`で `roach1` のコンソールに入れるので、メトリクスを眺めて楽しみましょう。

クエリ処理は roach1 のみ数値が上がり、CPU利用率は３つのノードに分散している様子を眺められたら完璧です。

ノードを消したり追加したりして、相互に状態監視が働き、スケールアウトができることや、障害に強いことを確かめてみましょう。
